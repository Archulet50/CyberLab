# IOC Reputation Checker

## Purpose

Checks indicators of compromise against a local training reputation database and assigns reputation, severity, source, and description.

## Features

- Classifies IP addresses, domains, MD5 hashes, and SHA-256 hashes
- Matches indicators against a local reputation database
- Assigns severity levels
- Tracks unknown indicators
- Generates a text report

## Usage

```bash
python ioc_reputation_checker.py
```

Press Enter to use the included `sample_iocs.txt`.
