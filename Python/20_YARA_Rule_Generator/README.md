# YARA Rule Generator

## Purpose

Creates a basic YARA rule from a list of suspicious strings.

## Features

- Reads text indicators
- Generates valid YARA syntax
- Saves `.yar` files
- Simple rule customization

## Usage

```bash
python yara_rule_generator.py
```

## Skills Demonstrated

- Python
- Detection Engineering
- Threat Hunting
- YARA Rule Creation

## Ethical Use

Generate rules only for authorized defensive security activities.
