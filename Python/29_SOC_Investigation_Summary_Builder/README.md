# SOC Investigation Summary Builder

## Purpose

Combines case, identity, IOC, authentication, MITRE ATT&CK, containment, and analyst recommendation data into a concise SOC investigation summary.

## Features

- Reads structured investigation data
- Validates required fields
- Summarizes login activity
- Includes IOC reputation and MITRE ATT&CK context
- Documents containment actions
- Produces an analyst assessment
- Generates a text report

## Usage

```bash
python soc_investigation_summary.py
