# IOC Scanner

## Purpose

Scans files for configured indicators of compromise, including suspicious filenames and known SHA-256 hashes.

## Features

- Recursive directory scanning
- SHA-256 file hashing
- Suspicious filename detection
- Configurable known-bad hashes
- Text report generation
- Permission and file-read error handling

## Usage

```bash
python ioc_scanner.py
