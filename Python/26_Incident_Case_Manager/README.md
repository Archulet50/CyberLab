# Incident Case Manager

## Purpose

Tracks and prioritizes security incident cases by severity and case status.

## Features

- Reads structured incident case data
- Sorts cases by severity and workflow status
- Displays active and closed cases
- Produces summary counts
- Generates text and CSV reports

## Usage

```bash
python incident_case_manager.py
```

Press Enter to use the included `sample_cases.txt`.

## Skills Demonstrated

- Python
- SOC Case Management
- Incident Response
- Severity Prioritization
- Workflow Tracking
- Text and CSV Reporting

## Limitation

This is a simplified training tool. Production case-management systems should also track assignees, timestamps, evidence, containment actions, remediation, and audit history.
