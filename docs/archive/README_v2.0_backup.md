# About CyberLab 

 

CyberLab is a professional cybersecurity portfolio and home laboratory built to demonstrate practical, hands-on security engineering skills. 

 

The project is developed primarily on **Ubuntu Linux** using **Python**, **Git**, and **GitHub**, with an emphasis on building tools that reflect real Security Operations Center (SOC) workflows. 

 

Rather than relying on tutorials alone, every project is designed, implemented, tested, documented, and version controlled to reinforce real-world cybersecurity concepts and provide tangible portfolio evidence.
